# Agrabah Logistics

```
git clone https://github.com/abaam/grabah-logistic-back-end
```

```
composer install
```

```
cp .env.example .env
```

```
php artisan key:generate
```

* Create your database and update the database configuration in your env

```
php artisan migrate
```

```
php artisan db:seed
```

```
php artisan serve
```
